# 105-从前序与中序遍历序列构造二叉树

## [题目](https://leetcode-cn.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/)(难度中等)

根据一棵树的前序遍历与中序遍历构造二叉树。

**注意**: 你可以假设树中没有重复的元素。

```markdown

例如，给出

前序遍历 preorder = [3,9,20,15,7]
中序遍历 inorder = [9,3,15,20,7]
返回如下的二叉树：

    3
   / \
  9  20
    /  \
   15   7

```
